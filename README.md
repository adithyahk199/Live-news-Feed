📰 Live News Feed – Async API Integration

A modern, responsive news aggregation application that delivers real-time news from a public News API. Built using HTML5, CSS3, and Vanilla JavaScript (ES6+), the project demonstrates best practices for asynchronous programming through the Fetch API and async/await, while providing a clean and user-friendly interface.

The application retrieves live news articles, dynamically renders content, and ensures a smooth user experience through loading indicators, comprehensive error handling, and interactive filtering options. It serves as a practical implementation of REST API integration and client-side asynchronous data processing.

⸻

✨ Key Features

* Real-Time News Retrieval using a public News API
* Asynchronous API Integration with async/await and the Fetch API
* Dynamic JSON Parsing and client-side rendering
* Keyword-Based Search for finding relevant news articles
* Category Filtering (Business, Technology, Sports, Entertainment, Health, Science, etc.)
* Loading Spinner to indicate ongoing API requests
* Robust Error Handling for network failures, invalid responses, and API limitations
* Responsive User Interface optimized for desktop, tablet, and mobile devices
* External Article Links directing users to the original news source
* Modern UI/UX with clean layouts and intuitive navigation

⚙️ Application Workflow

1. The user selects a news category or enters a search query.
2. An asynchronous request is sent to the News API using the Fetch API.
3. A loading indicator is displayed while awaiting the response.
4. The API response is validated and parsed as JSON.
5. News articles are dynamically generated and rendered as responsive cards.
6. Each article displays its title, image, description, source, publication date, and a link to the original article.
7. Any network or API errors are captured and presented through informative user-friendly messages.
